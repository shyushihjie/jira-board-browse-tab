# Privacy Policy

Jira Board Browse Tab does not collect, transmit, store on external servers, or sell personal data.

## Data Use

- The extension stores a small local configuration in Chrome storage:
  - Which Jira sites the user connected
  - Whether the behavior is enabled for each connected site

## Network Activity

- The extension does not send analytics, telemetry, or issue content to any third-party service.
- The extension does not execute remotely hosted code.

## Host Access

- The extension only runs on Jira sites that the user explicitly enables.
- Site access can be removed at any time from the extension options page or from Chrome's site access controls.
