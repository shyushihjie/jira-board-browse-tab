Chrome Extension

Jira Board Browse Tab

Open Jira board cards in dedicated /browse/ISSUE-KEY tabs instead of the board sidebar or modal.

What it does

  • Works on Jira Cloud board and backlog pages.
  • Requests host access only for Jira sites the user explicitly enables.
  • Stores site preferences locally in Chrome storage.
  • Does not use remote code, analytics, or third-party servers.
