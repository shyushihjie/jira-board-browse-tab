Privacy Policy

Jira Board Browse Tab

Last updated: April 22, 2026

Summary

Jira Board Browse Tab does not collect, sell, or transmit personal data to the developer or any third party. The extension stores a small amount of configuration locally in Chrome so it can remember which Jira sites you connected and whether the behavior is enabled for each site.

What the extension accesses

  • The current active tab URL, so the popup can determine whether the tab is a supported Jira board or backlog page.
  • Host access for Jira sites that you explicitly enable.
  • Local extension storage, so the extension can remember connected Jira sites and per-site enabled state.

What the extension stores

  • The Jira site origins that you connected.
  • Whether card interception is enabled for each connected Jira site.
  • Timestamps associated with those local settings.

This data is stored locally in Chrome extension storage and is removed when the extension is uninstalled. You can also remove site access at any time from the extension options page or Chrome's site access controls.

What the extension does not do

  • It does not send analytics, telemetry, or Jira issue content to any server.
  • It does not execute remotely hosted code.
  • It does not sell data, share data with advertisers, or use the extension for profiling.
