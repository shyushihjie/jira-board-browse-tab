# Store Listing Copy

## Name

Jira Board Browse Tab

## Short Description

Open Jira board cards in dedicated `/browse` issue tabs instead of the board sidebar.

## Detailed Description

Jira Board Browse Tab changes Jira Cloud board-card clicks so issues open in their own dedicated `/browse/ISSUE-KEY` tab instead of opening inside the board sidebar or dialog.

This keeps the board visible in one tab while giving each issue a stable full-page URL in another tab, which is especially useful when you want to review multiple tickets in parallel or keep issue pages pinned.

Key features:

- Open Jira board and backlog cards in dedicated issue tabs.
- Enable the behavior per Jira site instead of granting blanket access everywhere.
- Turn the behavior on or off later without disconnecting the site.
- Remove site access at any time from the extension options page.
- No remote code, no analytics, and no third-party services.

This extension is designed for Jira Cloud users who prefer full-page issue navigation over the default board sidebar workflow.
